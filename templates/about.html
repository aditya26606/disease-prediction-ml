Healthcare Data Analytics

About This Project

This system predicts diseases based on patient data using Machine Learning algorithms. It analyzes medical values and symptoms to assist in early disease detection.

🎯 Objectives

  • Provide fast disease prediction
  • Support healthcare decision making
  • Use data analytics for better diagnosis

🛠 Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python Flask
  • ML: Scikit-learn
  • Database: MySQL / SQLite
